Project Document Extractor

Drop Outlook .msg emails below. PDF and Excel attachments are scanned and key project fields are extracted into a spreadsheet.

How to use:
  1. Optionally configure your scope categories below
  2. Drag and drop your .msg files onto the zone (or click Browse files)
  3. Click Process & Download
  4. A .xlsx file downloads — one row per email — with all extracted project fields
⚙ Configure Scope Categories

One scope per line. The tool will search each document for these names and extract the associated percentage. These become columns in the output (e.g. "Concrete %").
